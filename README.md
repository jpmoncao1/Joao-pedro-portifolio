# Portfólio Profissional

Portfólio profissional de **João Pedro Monção Moreira**, desenvolvedor full stack com foco em interfaces modernas, produtos digitais e sistemas web funcionais.

## Sobre o projeto

Este portfólio foi desenvolvido para apresentar projetos reais, identidade profissional e capacidade técnica em diferentes tipos de solução digital.

O site reúne cases visuais e funcionais, com foco em:

- apresentação profissional
- navegação responsiva
- experiência de usuário
- clareza visual
- valorização dos projetos desenvolvidos

## Projetos em destaque

### Monarca Barber Club

Landing page vitrine para uma barbearia premium fictícia, com foco em posicionamento de marca, apresentação de serviços, diferenciais e contato.

**Tecnologias:**
HTML, CSS e JavaScript

### NexCRM

Sistema de CRM comercial desenvolvido de forma autoral, com dashboard executivo, gestão de leads, pipeline em kanban, agenda operacional e relatórios estratégicos.

**Tecnologias:**
React, Vite, JavaScript e Recharts

### Sweet Confeitaria

Dashboard administrativo autoral para gestão operacional de confeitaria, com visão de pedidos, produtos, faturamento, clientes e relatórios.

**Tecnologias:**
HTML, CSS e JavaScript

## Tecnologias utilizadas no portfólio
